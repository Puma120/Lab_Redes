ESP32 Dashboard

- Resumen -

Este proyecto implementa un sistema de monitoreo inteligente para el invernadero de la Universidad Iberoamericana mediante tecnología IoT basada en ESP32. El sistema registra parámetros ambientales críticos (humedad del suelo, temperatura/humedad del aire y luminosidad) y transmite los datos a un dashboard en tiempo real. La solución contribuye a los Objetivos de Desarrollo Sostenible (ODS) 2030, específicamente con el objetivo 11, al optimizar la gestión de recursos agrícolas y promover la innovación tecnológica en la institución educativa. Los resultados demuestran una mejora en la eficiencia del mantenimiento del invernadero y en la calidad del ambiente para el crecimiento de las plantas.

- Introducción -

Los invernaderos universitarios cumplen un papel fundamental en la investigación agrícola, la enseñanza y la producción alimentaria sostenible. Sin embargo, el monitoreo manual de las condiciones ambientales resulta ineficiente y propenso a errores. Este proyecto nace de la necesidad de optimizar la gestión del invernadero de la Universidad Iberoamericana mediante tecnologías de la información.

La Agenda 2030 para el Desarrollo Sostenible establece la necesidad de implementar tecnologías que contribuyan a la seguridad alimentaria y al fortalecimiento institucional mediante la innovación. En este contexto, la implementación de sistemas IoT representa una oportunidad para mejorar la eficiencia y sostenibilidad de los espacios agrícolas educativos.

El objetivo principal de este proyecto es desarrollar e implementar un sistema automatizado de monitoreo para el invernadero universitario que

-Registrar en tiempo real los parámetros ambientales críticos para el desarrollo vegetal
-Facilitar el acceso a los datos desde cualquier lugar mediante un dashboard web
-Contribuir a la optimización de recursos como agua y energía
-Servir como herramienta educativa para estudiantes de agronomía y tecnología

- Materiales -

- Node MCU ESP32

- Caja impresa en 3D

- Sensor Dht22

- Sensor de Humedad de tierra

- Sensor LDR (Fotoresistencia)

- Protoboard y cables

- Desarrollo -

El primer paso consistió en identificar los parámetros críticos para el monitoreo del invernadero, determinando tres variables fundamentales:

1- Humedad del suelo: factor crítico para la gestión del riego
2- Temperatura y humedad del aire: condicionantes del desarrollo vegetal
3- Luminosidad: esencial para la fotosíntesis y crecimiento

- Resultados -

La implementación del sistema ha reducido el tiempo dedicado al monitoreo manual y ha optimizado el uso de agua.

El sistema se ha integrado en tres plantaciones y ha facilitado el crecimiento y cuidado de las plantas de los mismos.

- Conclusiones -

El sistema de monitoreo IoT ha demostrado ser una solución efectiva y sostenible, con el ESP32 como plataforma robusta para aplicaciones agrícolas.

El proyecto contribuye directamente al ods 11, proporciona valor educativo significativo y presenta oportunidades de escalabilidad hacia un invernadero completamente automatizado.

Los principales desafíos superados fueron la conectividad en zonas con señal débil y la protección de componentes electrónicos en ambientes húmedos.
Como trabajo futuro, se contempla la integración de actuadores para control automático de las condiciones ambientales.
